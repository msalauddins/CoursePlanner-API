Welcome
This documentation will help you to use the Course Planner API

This documentation is to help you regarding each step of customization. Please go through the documentation carefully to understand how this API is made and how to add or edit properly.

Requirements

You will need the following sofwares/plugins/dependencies to run this API.

  1. Code Editing Software (eg: Sublime Text or Visual Studio Code)
  2. Web Browser for testing (eg: Google Chrome or Safari and Postman)
  3. dependencies:
    ● auth0-js
    ● body-parser
    ● chai-http
    ● express
    ● express-prettify
    ● jsonwebtoken
    ● mocha
    ● mongod
    ● mongoose
    ● npm
    ● request
    ● should
    ● update

Be careful while installing the dependencies.

Overview#back to top

This section contain an overview of the data provided and the API purpose.

Using the API

We built the API to be as self-documenting as possible.
But if you find yourself overwhelmed, we organized this site into four major areas.

1. Getting started introduces you to the operations offered by the API.
2. API calls gives you a hands-on experience of those operations with an interactive console.
3. Field reference lists and describes the type of information provided by the API.
4. Contact Us If you need support or need to get in touch, this is the place to look.

Endpoints

Endpoint What it does
/areas Returns all Area from Database
/courses Returns all Course name from Database
/users Returns all User names from Database
/find/areas Single string query request. Returns Area names from Database according to "areaid"
/find/course Single string query request. Returns Area names from Database according to "courseid"
/find Multiple string query request. Returns Area names from Database according to "areaid" & "courseid"
/status Check server status
/add Returns a html form to create new user
/delete Returns a html form to delete a user
/update Returns a html form to update a user
/new/user/dialog Add new user to the database
/userid/:id Delete user by using "id"
/delete/user/:uid Delete user by using "uid"
/update/user/:uid Find user by "uid" and update any data
/login Returns HTML login form
/log-in Login using "uid" and "pass"
/home Returns HTML Home page
/jwt Login using "uid" and "pass" with a secure Json Web Token
/jwthome Returns HTML Secure Home page
